CRM-Next-GraphQL

Este proyecto es un CRM administrador de clientes, esta creado con Next.js, GraphQL, Node.js, Apollo server, como base de datos MongoDB, para crear una cuenta nueva se debe entrar a la url https://crm-next-jm.netlify.app/nuevacuenta, cada vendedor tiene sus propios clientes y no puede acceder a los clientes de otro vendedor
